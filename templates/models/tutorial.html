How to search for models?

Under Explore the models, you can type in your keywords in the different search fields.

If you press "Search", the list will only show results that fit your search query.

How to add a model?

If you go to "submit a model", you can login here.

This will take you to the backend of the website. On the the left-hand side, you'll see a menu with the heading "MODELS". Below this heading, you'll see several menu items.

To add a model, press the "+ Add" to the right of the item "Psychomodels". Then do the following:

  • Submitting user = select your username from the list
  • Reviewed [] = check this box
  • Publication = press "+ Add" to add a publication. This opens a new window
    • Unique identifier = add DOI or journal page URL
    • Title = paste title of the publication in which to model has appeared here
    • Model author = select the first author of the publication or press "+" to add this author. A new window will appear. What is requested should be self explanatory.
    • Press "SAVE"
  • Model name = give the model an appropriate and uniquely identifying name
  • Description = provide a short paragraph describing the model, which answers the three questions:
    • What is it about?
    • What does it do?
    • How does it work?
  • Language = select the programming language in which the model was written. If the language is not available in the list, press "+". Another window will open:
    • Language name = give language name
    • Language description = provide an URL to a webpage that contains documentation about the language.
    • Press "SAVE"
  • Framework = select the framework or model type to which the model belongs. If it is not listed, select "Other" or add your own.
  • CodeURL = if available, give a URL of the online storage of the model code
  • DataURL = if the model uses data and this data is available online, add the URL to these data here.

Press "SAVE"
